Do it easy - perform automated Gambio to WooCommerce migration with the help of Cart2Cart plugin.
== Description ==

Cart2Cart: Gambio to WooCommerce Migration plugin will take the steering wheel of data transfer in its hands. It offers a great possibility to move from one platform to another with no programming skills or expertise required. Establish a connection between your stores, choose the entities you want to transfer, adjust the migration to your needs with the help of additional options and let it flow. Note, although  the plugin is free to download, migration itself is paid and starts at $69.

= Features =
* **Flexible Price.** Cart2Cart makes migration affordable. Its price depends on the amount of transferring entities and additional options chosen.
* **100% Uptime.** Your current Gambio store won’t be affected by migration process in any way - it’ll be available for your customers as always. 
* **Professional Support.** Cart2Cart qualified Support Team will give you a hand in case you’ve encountered a problem or got any questions. 
* **Demo Migration.** Not sure about the transfer? Test how Cart2Cart plugin works by moving up to 10 entities to WooCommerce absolutely for free with the help of Demo Migration option.

**Transfer the following data from Gambio to WooCommerce:**

* Products, product images, product extra fields, product attributes, product variants;
* Categories, category images;
* Customers, customer shipping address, customer billing address;
* Orders, order statuses, order products;
* Manufacturers, taxes, reviews.

*Supported WooCommerce versions:* 1.1.x - 1.6.x, 2.x
**Please note:** Cart2Cart migrate data, not design. Your current Gambio website design will be lost.

In order to begin migration, you have to download the migration plugin, install it and establish the connection between your Gambio and WooCommerce stores. Afterwards, you’ll be redirected to Cart2Cart website to proceed with the migration. To make sure everything will work out okay, choose free Demo Migration option, that will move up to 10 entities to WooCommerce in terms of minutes. If you’re satisfied with the results of Demo Migration, select Insurance plan (not obligatory) and launch Full Migration to start an automated data transfer. 

**Pre-Migration Steps**

1. Install WooCommerce store and make sure it be reached via internet.
1. Check for FTP details for both of your shopping carts. They are required to perform migration. 

== Installation ==

1. Download plugin.zip file and extract it to your hard drive
1. Upload unzipped file to wp-content/plugin directory
1. Activate “Cart2Cart Gambio to WooCommerce Migration” in plugins section of your admin panel
1. Insert FTP details to upload the Connection Bridge files for Gambio 
1. Connection Bridge files for WooCommerce are will be uploaded automatically
1. Click Start Migration button and follow the Migration Wizard to perform accurate and swift migration

== Frequently Asked Questions ==

= Will customers’ credit card information be migrated? =
Unfortunately, clients' credit card information cannot be migrated. Though, the information about customers' billing, shipping addresses will be successfully moved.

= Is it possible to migrate to the same domain on a different directory? =
Yes, it is. You can migrate within same domain using different folders.

= Does service migrate SEO? =
Unfortunately, SEO data is not moved during Gambio migration. Therefore, be ready that store’s URLs and structure will be modified.
= Is there a possibility to migrate reviews to WooCommerce? =
Yes, Cart2Cart supports migration of reviews while moving to and from WooCommerce. Moreover, review ratings will be moved together with reviews.
